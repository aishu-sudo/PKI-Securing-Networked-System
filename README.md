# 🔐 Securing a Networked System with PKI

## Project Overview

This project demonstrates the implementation of a secure HTTPS web server using Public Key Infrastructure (PKI) on Kali Linux.

The system uses a complete certificate trust chain consisting of:

- Root Certificate Authority (Root CA)
- Subordinate Certificate Authority (AcmeCA)
- Server Certificate

to establish secure communication.

---

## Features

- ✅ Public Key Infrastructure (PKI) Implementation
- ✅ Root CA and Subordinate CA Configuration
- ✅ SSL/TLS Certificate Generation and Signing
- ✅ Apache2 HTTPS Server Deployment
- ✅ DNS Configuration
- ✅ UFW Firewall Security Rules
- ✅ Secure File Upload Portal
- ✅ Browser Certificate Validation
- ✅ TLS Handshake Analysis using Wireshark

## PKI Architecture

```text

   │
   ▼
```
